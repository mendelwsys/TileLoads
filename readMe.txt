Загрузчик тайлов.
Загружает тайлы и делает из них файл c расширением *.mbtiles
Примеры запуска и описание параметров в startLoader.js

+Производится фильтрация тайлов по полигону. 
+Добавлена обработка таймаутов при запросе, проверено на работоспособность отсутсвии интернет.
+Усовершенстована обработка ошибок, что бы избежать задваивания запросов и потери тайлов при таймаутах
+Добавлены журналы ожидаемого ко-ва тайлов на каждую попытку скачивания, используется для сравнения с скаченным фактическим кол-вом  
+Добавлено обработка contentType, и анализ входящего потока данных, теперь все что отличается от изобрадения накапливается в отдельной папке

-Не обрабатываются архивы, соответсвенно не нужно указывать в заголовке Accept-Encoding, что вы принимаете архивы,если сервер вернет архив он будет сохранен в папке ошибочных ответов.
-Для текущей версии фильтрация работает до 110 000 000  тайлов, 
(надо установить --max-old-space-size=8192 при запуске)
Предел тайлов превышается для примера в startLoader при 21-м масштабе.
